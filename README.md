# LP診断フォーム（lp-diagnosis）

このリポジトリは、個人事業主・ひとり経営者向けLPに連動する  
**無料Web導線診断フォーム**です。

---

## 概要
- 7つの質問に回答することで、Web導線の状態を簡易診断します
- 診断結果は以下の4段階で表示されます  
  - complete  
  - warning  
  - alert  
  - critical
- complete 以外の結果では、**無料相談（Googleフォーム）への導線**を表示します
- 診断結果はページ内に即時表示され、メール送信などは行いません

---

## ファイル構成
lp-diagnosis/
├─ index.html # 診断フォーム本体
├─ style.css # LPと世界観を揃えたCSS
└─ script.js # 診断ロジック・結果判定用JS

---

## 使い方
1. 本リポジトリを GitHub Pages で公開
2. LP内のCTAから診断フォームへリンク
3. ユーザーが診断に回答
4. 結果表示 → 必要に応じて無料相談フォームへ誘導

---

## 注意事項
- 本診断はあくまで簡易的なチェックです
- 自動メール送信や個人情報の保存は行いません
- 無料相談はリンク先のGoogleフォームから受け付けます
